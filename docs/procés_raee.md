# Protocol Intern de Gestió de RAEE

A SmartBench seguim aquest procediment cada vegada que un component (disc dur, RAM, monitor, etc.) s'espatlla o ja no funciona per a l'anàlisi d'IA.

### Passos del protocol:

1. **Detecció i Retirada:** Es retira l'equip que no funciona o ha quedat obsolet.
2. **Esborrat Segur de Dades (CRÍTIC):** Passem un software de destrucció de dades a tots els discos durs per seguretat.
3. **Avaluació de la "Segona Vida":**
    * **Prioritat 1 - Reutilitzar:** Es pot fer servir per a una tasca més senzilla (com fer factures)?
    * **Prioritat 2 - Reparar:** Es pot arreglar canviant una peça petita?
    * **Prioritat 3 - Donar/Vendre:** Si funciona però no el necessitem, el donem a una ONG o el venem com a reacondicionat.
4. **Reciclatge Final:** Si l'equip és ferralla, el portem al gestor autoritzat o a la deixalleria.
5. **Certificació:** Guardem el comprovant de reciclatge per al nostre registre de sostenibilitat.

**Exemple aplicat:** Si se'ns trenca un portàtil de SmartBench, mirem si la pantalla funciona per usar-la com a monitor extern abans de portar-lo a reciclar.
