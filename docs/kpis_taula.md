# Taula d'Indicadors (KPIs) de SmartBench

Aquests són els controls que fem mensualment per saber si estem complint els nostres objectius de sostenibilitat.

| Indicador | Què mesura | Com es calcula | Freqüència | Objectiu |
| :--- | :--- | :--- | :--- | :--- |
| **Consum Energètic** | L'electricitat dels servidors. | kWh totals mesurats al panell del Cloud. | Mensual | Reduir un 10% anual. |
| **Taxa de Reutilització** | L'ús de hardware no nou. | (Equips reusats / Equips totals) x 100. | Trimestral | Mínim el 40% del hardware. |
| **Eficiència de Codi** | Temps de procés per vídeo. | Segons de CPU per cada partit analitzat. | Setmanal | Millorar un 5% l'eficiència. |
| **Gestió de Residus** | El reciclatge correcte. | kg de RAEE enviats a gestor autoritzat. | Semestral | 100% de residus certificats. |
